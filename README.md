# Clipboard Watcher

```sh
cargo +nightly install --git https://github.com/ferreira-tb/clipboard-watcher
```

|     Key     | Description            |
| :---------: | :--------------------- |
|     `F`     | Flush the cache.       |
|     `Q`     | Flush, then exit.      |
|     `X`     | Exit.                  |
| `Backspace` | Remove the last entry. |
|  `Delete`   | Remove all entries.    |
|   `Enter`   | Toggle the watcher.    |
